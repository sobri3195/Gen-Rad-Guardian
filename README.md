# Gen-Rad Guardian

Gen-Rad Guardian is a modern, responsive preventive cancer risk management web application built with React, Vite, TypeScript, Tailwind CSS v4, shadcn/ui patterns, and lucide-react.

## What it includes

- Landing page with preventive positioning, trust messaging, and CTA flow.
- Dashboard-focused application shell with sidebar, topbar, and mobile navigation.
- Risk assessment dashboard, profile, screening, trends, education, explainability, and population insights pages.
- Reusable cards for risk scores, factors, alerts, trends, recommendations, explainability, cohort insights, dataset status, loading states, and empty states.
- Mock datasets that demonstrate user-level and cohort-level preventive oncology workflows.
- Responsive health-tech visual design suitable for a startup MVP, demo, or innovation showcase.
- Clear disclaimers that outputs are supportive and educational only and require professional clinical validation.
- Vercel-ready Vite deployment structure.

## Tech stack

- React
- Vite
- TypeScript
- Tailwind CSS v4
- shadcn/ui-style component structure
- lucide-react
- React Router
- Recharts

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Deployment

This project is ready to deploy to Vercel as a standard Vite application.

## Safety notice

Gen-Rad Guardian is **not** a diagnosis system, **not** a replacement for physicians, and **not** a definitive treatment recommendation engine. All outputs are supportive, educational, and must be validated by qualified healthcare professionals.
